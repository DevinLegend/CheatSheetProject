TABLE TAGS

Tags Name Description Notes
<table> Table The wrapper element for all HTML tables. Every table element goes inside this
<thead> Table Head The set of rows defining the column headers in a table. table -> thead same level as tbody
<th> Table Heading The set of headings defining the head in a table. thead -> tr -> th
<tbody> Table Body The set of rows containing actual table data. (table -> tbody)(same level as thead)
<tr> Table Row The table row container. (inside thead, tbody, & tfoot)
<td> Table Data The table data container. tr -> td
<tfoot> Table Foot The set of rows defining the footer in a table. Same level as tbody and thead

TABLE ATRRIBUTES

Attribute Name Description Notes
<colspan> Column Span Defines how many columns a td element should span. Column = horizontal
<rowspan> Row Span Defines how many rows a td element should span. row = verticle
